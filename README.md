# Bachat Gat Software

## 📌 Overview

Bachat Gat Software is a web-based application developed to manage and monitor the activities of Saving Groups (Self-Help Groups). The system helps administrators maintain member records, savings, loans, repayments, meetings, and financial reports efficiently.

The project was developed using Java Servlets, JDBC, MySQL, HTML, CSS, and JavaScript.

---

## 🚀 Features

### 👥 Group Management

* Create and manage saving groups
* Store and update group information
* View group details

### 👤 Member Management

* Add new members
* View member details
* Update member information
* Delete member records

### 🏦 Bank Management

* Maintain bank account details
* Store bank-related information
* Manage financial records

### 💰 Loan Management

* Add internal loan details
* Track loan information
* Monitor loan distribution

### 💳 Loan Repayment Management

* Record repayment details
* Track installments
* Manage repayment history

### 📅 Meeting Management

* Schedule meetings
* Store meeting records
* Maintain meeting history

### 📊 Financial Reports

* Generate financial summaries
* Track group savings
* Monitor loan and repayment activities
